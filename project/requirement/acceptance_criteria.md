# Supreme Tuning – Project Acceptance Criteria

The project is considered completed when:

### 1. Backend
- All endpoints return correct JSON
- Admin save writes to JSON
- JSON backup works

### 2. Scraper
- All German brands + Lamborghini/Bentley/Aston imported
- Stage 1 & 2 extracted
- Stage 1+ & 2+ generated
- BMW & AMG rules applied

### 3. Admin Panel
- Fully editable fields
- Saves persist on reload
- Protected behind login

### 4. Calculator
- Brand → Model → Engine → Stage flow fully working
- Tuned HP/Nm + gains calculated
- ECU unlock & CPC rules visible
- Shown on frontend without errors

### 5. Deployment
- Backend/API accessible
- Frontend accessible
- Both work on staging/production

Only when all conditions are met is the milestone considered delivered.
